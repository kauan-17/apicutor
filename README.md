# Apicutor - Sistema de Gerenciamento para Apicultura

## Descrição
Apicutor é um sistema completo para gerenciamento de apiários, colmeias e produção apícola, desenvolvido com Spring Boot no backend e Angular no frontend.

## Requisitos
- Java 11+
- Node.js 14+
- NPM 6+
- Maven 3.6+

## Executando o Sistema

### Backend
1. Navegue até a pasta do backend:
```
cd backend
```

2. Execute o projeto Spring Boot:
```
mvn spring-boot:run
```
O backend estará disponível em http://localhost:8080

### Frontend
1. Navegue até a pasta do frontend:
```
cd frontend
```

2. Instale as dependências:
```
npm install
```

3. Execute o projeto Angular:
```
npm start
```
O frontend estará disponível em http://localhost:4200

## Funcionalidades Principais
- Gerenciamento de apiários
- Controle de colmeias
- Registro de inspeções
- Acompanhamento de rainhas
- Controle de produção
- Autenticação de usuários

## Usuário Padrão
- Username: admin
- Senha: admin123

## Tecnologias Utilizadas
- Backend: Spring Boot, Spring Security, Spring Data JPA, H2 Database
- Frontend: Angular, Bootstrap
- Autenticação: JWT
